# **BringItOn**

BringItOn is a simple productivity and accountability app I’m building with React, TypeScript, Vite, TailwindCSS, and Supabase.  
The goal is to create a clean, fast, and minimal tool that helps people stay consistent and track their progress.

---

## **Features**

- User authentication (Supabase)
- Create and manage tasks
- Fast development setup with Vite
- Responsive UI with TailwindCSS
- Client-side routing with React Router
- More features coming as the project grows

---

## **Tech Stack**

- **React + TypeScript**
- **Vite**
- **TailwindCSS**
- **Supabase**
- **React Router**
- Zustand (planned)

## **Getting Started**

Clone the repo:

```bash
git clone https://github.com/amrituk7/bringiton.git
cd bringiton
```

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```
